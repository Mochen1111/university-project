线上超市购物系统
===========================

###########环境依赖

1. JDK 1.8

2. MySQL 8.0

###########部署步骤
1. 修改数据库连接配置
    
    位于application.properties文件下

2. 修改文件上传路径

    位于application.properties文件下的dirPath（文件路径最后必须带有 \ ）
